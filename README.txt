Проект store имеет следующую структуру:

    Абстрактный класс Item представляет собой наименование определенного
продукта в магазине. В классе имеется абстрактный метод
cost(int numberOfItem), который возвращает цену за определенное
количество определенного продукта. Классы-наследники - A, B, C, D -
переопределяют данный метод.

    Абстрактный класс SpecialOffer, расширяющий класс Item,
представляет собой специальные акционные предложения.
Классы OfferThreeForThree и OfferFiveForSix расширяют класс
SpecialOffer и представляют собой акционные предложения
допустим, 3$ за 3 единицы продукта и 5$ за 6 единиц продукта
соответственно. В классах переопределен метод cost(int numberOfItem),
позволяющий рассчитать стоимость данного количества определенного
продукта с учетом акции. Класс-акция - "декоратор" для продукта.

    Класс GroceryStore представляет собой продуктовый магазин.
Класс предоставляет метод для получения общей стоимости "корзины"
calculateTotalCost(String cartContent) в зависимости от ее
наполнения - строки, что передается в качестве аргумента мотоду.
Переданная строка проверяется на корректность. В случае, если
строка null или пустая, генерируется собственное
исключение GroceryStoreException, которое выводит определенное
сообщение об ошибке. Также генерируется исключение, если
аргумент не содержит ни одного валидного символа "A", "B", "C"
или "D". Если строка содержит хотя бы один валидный символ,
стоимость корзины рассчитывается, игнорируя переданные
невалидные символы.
    Класс GroceryStore не предназначен для наследования, поэтому
объявлен финальным.

    Приложение также содержит тестовые классы GroceryStoreTest,
OfferFiveForSixTest, OfferThreeForThree.
